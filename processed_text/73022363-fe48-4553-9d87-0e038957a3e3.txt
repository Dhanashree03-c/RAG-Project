What is Bootstrap?
Bootstrap is a Framework toolkit. It works same as CSS. It works on the same concept of Flexbox in CSS. Bootstrap has many pre-built components like button, forms, navbar, card, etc. Bootstrap is a powerful, feature-packed frontend toolkit. Build anything – from prototype – in minutes. It contains pre-built CSS files like style.css from which we can import any style. 
Benefits of Bootstrap:
Makes Development Faster & Easier
Good choice of pre-built components
good browser Compatibility
